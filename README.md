Expense Tracker (Python)
Description : 

This is a simple Expense Tracker project built using Python.
It allows users to add, view, and calculate their daily expenses in different categories.

Features
Add new expenses
View all expenses
Calculate total expenses


 Technologies Used :
Python (Core Python concepts)


 How to Run
 
1.Install Python on your system
2.Download or clone this repository
3.Open terminal or command prompt
4. Right Click ---> Run File in Terminal 


Run the file:

python expense tracker.py

1. Add Expense
2. View Expenses
3. Total Expense
4. Exit
